# Security Policy

## Supported versions

FunkHub is in active early development. Report security issues for the latest release and current `main` branch.

## Reporting a vulnerability

Please do not open public issues for security vulnerabilities.

Instead:

1. Open a private security advisory on GitHub (preferred), or
2. Contact maintainers privately via Discord: https://discord.gg/cdP7JhDv4u

Include:

- Affected version/commit
- Reproduction steps or proof of concept
- Impact assessment
- Suggested mitigation (if known)

We will acknowledge reports as soon as possible and coordinate a fix/release.

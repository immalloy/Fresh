# Code of Conduct

## Our pledge

We are committed to making participation in this project a respectful, harassment-free experience for everyone.

## Our standards

Examples of behavior that contributes to a positive environment:

- Being respectful and constructive
- Accepting feedback gracefully
- Focusing on what is best for the project and community

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks
- Trolling or deliberately disruptive behavior
- Publishing private information without permission

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct and may remove, edit, or reject comments, commits, code, issues, and PRs that violate it.

## Reporting

Report concerns privately to project maintainers through GitHub or Discord:

- https://discord.gg/cdP7JhDv4u

## Scope

This Code of Conduct applies within project spaces and public spaces when representing the project.
